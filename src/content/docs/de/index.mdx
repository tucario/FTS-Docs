## Unterstützte Objekte

- **Standardobjekte**: Account, Contact, Case, Lead, Opportunity, Campaign, Order
- **Benutzerdefinierte Objekte**: Jedes benutzerdefinierte Objekt mit Private oder Public Read Only OWD

## Schnellstart

1. **Installieren** Sie das Paket aus AppExchange
2. **Weisen Sie** Permission Sets zu (FTS_App_Access für Administratoren, FTS_Data_Access für Benutzer)
3. **Konfigurieren** Sie Objekte über den Configuration Wizard
4. **Fügen Sie** die Team Member-Komponente zu Datensatzseiten hinzu

Bereit anzufangen? Gehen Sie zum [Installationsleitfaden](/de/getting-started/installation/).
