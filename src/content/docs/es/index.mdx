## Objetos Soportados

- **Objetos Estándar**: Account, Contact, Case, Lead, Opportunity, Campaign, Order
- **Objetos Personalizados**: Cualquier objeto personalizado con OWD Privado o Público de Solo Lectura

## Inicio Rápido

1. **Instale** el paquete desde AppExchange
2. **Asigne** Permission Sets (FTS_App_Access para administradores, FTS_Data_Access para usuarios)
3. **Configure** objetos mediante el Asistente de Configuración
4. **Agregue** el componente Miembro de Equipo a las páginas de registro

¿Listo para comenzar? Diríjase a la [Guía de Instalación](/es/getting-started/installation/).
