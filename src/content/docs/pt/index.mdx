## Objetos Suportados

- **Objetos Padrão**: Account, Contact, Case, Lead, Opportunity, Campaign, Order
- **Objetos Personalizados**: Qualquer objeto personalizado com OWD Private ou Public Read Only

## Início Rápido

1. **Instale** o pacote do AppExchange
2. **Atribua** permission sets (FTS_App_Access para admins, FTS_Data_Access para usuários)
3. **Configure** objetos através do Configuration Wizard
4. **Adicione** o componente Team Member às páginas de registro

Pronto para começar? Acesse o [Guia de Instalação](/pt/getting-started/installation/).
