## Obsługiwane obiekty

- **Obiekty standardowe**: Account, Contact, Case, Lead, Opportunity, Campaign, Order
- **Obiekty niestandardowe**: Dowolny obiekt niestandardowy z OWD typu Private lub Public Read Only

## Szybki start

1. **Zainstaluj** pakiet z AppExchange
2. **Przypisz** Permission Sets (FTS_App_Access dla administratorów, FTS_Data_Access dla użytkowników)
3. **Skonfiguruj** obiekty za pomocą Configuration Wizard
4. **Dodaj** komponent Team Member do stron rekordów

Gotowy, aby rozpocząć? Przejdź do [Przewodnika instalacji](/pl/getting-started/installation/).
