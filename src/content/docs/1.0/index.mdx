## Key Features

<CardGrid stagger>
  <Card title="Universal Team Membership" icon="group">
    Add team members to any object, not just Accounts. Supports standard and custom objects with Private or Public Read Only OWD.
  </Card>

  <Card title="Automatic Record Sharing" icon="approve-check-circle">
    Team members automatically receive appropriate access. Share records are created, updated, and removed automatically.
  </Card>

  <Card title="Role-Based Access" icon="setting">
    Owner, Manager, and User roles with distinct permissions. Managers can add/edit/remove team members.
  </Card>

  <Card title="Temporary Assignments" icon="clock">
    Time-limited team memberships with automatic cleanup. Set an end date and expired members are removed automatically.
  </Card>
</CardGrid>

## Supported Objects

* **Standard Objects**: Account, Contact, Case, Lead, Opportunity, Campaign, Order
* **Custom Objects**: Any custom object with Private or Public Read Only OWD

## Quick Start

1. **Install** the package from AppExchange
2. **Assign** permission sets (FTS\_App\_Access for admins, FTS\_Data\_Access for users)
3. **Configure** objects via the Configuration Wizard
4. **Add** the Team Member component to record pages

Ready to begin? Head to the [Installation Guide](/1.0/getting-started/installation/).
