## حالة الاستخدام 8: الوصول بدون Permission Set

### الهدف

التحقق من السلوك عندما يفتقر المستخدم للأذونات المطلوبة.

### المتطلبات الأساسية

- مستخدم بدون Permission Set FTS_Data_Access
- صفحة سجل مع مكون Object Team

### الخطوات

| الخطوة | الإجراء | النتيجة المتوقعة |
|------|--------|----------------|
| 1 | افتح صفحة السجل | تحميل الصفحة |
| 2 | عرض مكون Object Team | عرض رسالة خطأ |

![خطأ Access Required](../../../../assets/pictures/access-required.png)

### نقاط التحقق

- [ ] رسالة خطأ واضحة حول الوصول المفقود
- [ ] عدم عرض أي بيانات
- [ ] عدم وجود أخطاء/استثناءات نظام
