## الكائنات المدعومة

- **الكائنات القياسية**: Account، Contact، Case، Lead، Opportunity، Campaign، Order
- **الكائنات المخصصة**: أي كائن مخصص مع Private أو Public Read Only OWD

## البدء السريع

1. **التثبيت** من AppExchange
2. **التعيين** لمجموعات الأذونات (FTS_App_Access للمسؤولين، FTS_Data_Access للمستخدمين)
3. **التكوين** للكائنات عبر معالج التكوين
4. **الإضافة** لمكون عضو الفريق إلى صفحات السجلات

هل أنت مستعد للبدء؟ انتقل إلى [دليل التثبيت](/ar/getting-started/installation/).
