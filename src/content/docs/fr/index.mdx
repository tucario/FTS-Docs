## Objets pris en charge

- **Objets standard** : Account, Contact, Case, Lead, Opportunity, Campaign, Order
- **Objets personnalisés** : Tout objet personnalisé avec OWD privé ou lecture publique seule

## Démarrage rapide

1. **Installez** le package depuis AppExchange
2. **Attribuez** les ensembles de permissions (FTS_App_Access pour les administrateurs, FTS_Data_Access pour les utilisateurs)
3. **Configurez** les objets via l'assistant de configuration
4. **Ajoutez** le composant Membre d'équipe aux pages d'enregistrement

Prêt à commencer ? Consultez le [guide d'installation](/fr/getting-started/installation/).
