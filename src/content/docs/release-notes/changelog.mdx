## v1.0 — 2026-02

Initial release of Flexible Team Share.

### Added

- **Universal Team Membership** — Add team members to any Salesforce object (standard and custom)
- **Automatic Record Sharing** — Share records created/updated/deleted automatically based on team membership
- **Role-Based Access** — Owner, Manager, and User roles with distinct permissions
- **Temporary Assignments** — End date support with automatic cleanup via scheduled batch job
- **Owner Sync** — Invocable Action (`SyncOwnerInvocable`) to keep team Owner in sync when parent record owner changes
- **Configuration Wizard** — Admin UI to enable team sharing per object
- **Collapsible Team List** — Configurable display limit with "Show more/less" toggle
- **Customizable Component** — Card title and max displayed members configurable in Lightning App Builder
- **Multi-Object Support** — Account, Contact, Case, Lead, Opportunity, Campaign, Order, and any custom object
- **Internationalization** — 35 languages supported via Custom Labels and Translation Workbench
- **Permission Sets** — FTS_App_Access (admin) and FTS_Data_Access (user)
- **Security** — CRUD/FLS enforcement, SOQL injection prevention, sharing model compliance

See the full [v1.0 release notes](/FTS-Docs/release-notes/1-0/) for details.
