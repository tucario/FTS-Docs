## Supported Objects

- **Standard Objects**: Account, Contact, Case, Lead, Opportunity, Campaign, Order
- **Custom Objects**: Any custom object with Private or Public Read Only OWD

## Quick Start

1. **Install** the package from AppExchange
2. **Assign** permission sets (FTS_App_Access for admins, FTS_Data_Access for users)
3. **Configure** objects via the Configuration Wizard
4. **Add** the Team Member component to record pages

Ready to begin? Head to the [Installation Guide](/FTS-Docs/getting-started/installation/).
