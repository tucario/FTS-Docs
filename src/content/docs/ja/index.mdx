## サポートされているオブジェクト

- **標準オブジェクト**: Account、Contact、Case、Lead、Opportunity、Campaign、Order
- **カスタムオブジェクト**: PrivateまたはPublic Read OnlyのOWDを持つ任意のカスタムオブジェクト

## クイックスタート

1. **インストール** AppExchangeからパッケージをインストール
2. **割り当て** Permission Setを割り当て（管理者にはFTS_App_Access、ユーザーにはFTS_Data_Access）
3. **設定** Configuration Wizardを使用してオブジェクトを設定
4. **追加** レコードページにチームメンバーコンポーネントを追加

準備はできましたか？[インストールガイド](/ja/getting-started/installation/)にお進みください。
