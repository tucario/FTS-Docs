## ユースケース8：Permission Setなしのアクセス

### 目的

ユーザーが必要な権限を持っていない場合の動作を確認します。

### 前提条件

- FTS_Data_Access Permission Setを持っていないユーザー
- Object Teamコンポーネントを含むレコードページ

### 手順

| ステップ | アクション | 期待される結果 |
|------|--------|----------------|
| 1 | レコードページを開く | ページが読み込まれる |
| 2 | Object Teamコンポーネントを表示 | エラーメッセージが表示される |

![Access Required error](../../../../assets/pictures/access-required.png)

### 検証ポイント

- [ ] アクセス不足に関する明確なエラーメッセージ
- [ ] データが公開されていない
- [ ] システムエラー/例外がない
